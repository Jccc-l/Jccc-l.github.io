虚拟环境创建

对多版本的Python进行管理，一般使用虚拟环境

创建虚拟环境可以通过python3.x -m venv venv（python3.x替换为你所需的Python版本）命令进行。

创建好虚拟环境以后，通过source venv/bin/activate命令将虚拟环境激活

随后即可使用pip命令在虚拟环境中管理第三方库文件。

如果想退出虚拟环境，输入deactivate命令。
